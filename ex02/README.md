## 電卓プログラム (ex02/calc.py)
### 追加機能
- 新たな演算 : +,-以外の四則演算を実装
- クリア : textboxに入力されたデータを１文字削除する
- オールクリア : textboxに入力されたデータをすべて削除する
- ランダム数字選択 : Rのボタンを押すと、ランダムで0から9のランダムな数字が表示される
- .＆x10ボタン : .とx10のボタンを押すと、1/10,10倍されるボタン
